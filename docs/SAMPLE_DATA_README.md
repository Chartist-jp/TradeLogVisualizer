# TradeLog Visualizer - サンプルCSVファイル

このディレクトリには、アプリケーションのテスト用サンプルCSVファイルが含まれています。

## ファイル一覧

### sample_jp_trades.csv
日本株のサンプルデータです。以下の取引が含まれています：
- ソフトバンクグループ (9984) - 買い・売り
- トヨタ自動車 (7203) - 買い

### sample_us_trades.csv
米国株のサンプルデータです。以下の取引が含まれています：
- AMTM (New York Stock Exchange) - 買い・売り
- AAPL (NASDAQ) - 買い

## 使用方法

1. アプリケーションを起動します (`npm run dev`)
2. サンプルCSVファイルをドラッグ&ドロップ、またはファイル選択ボタンでアップロード
3. **日本株・米国株は自動判別されます**（手動選択は不要）
4. インポートされたデータが下部のテーブルに表示されます

## 自動判別の仕組み

CSVファイルの内容から以下の特徴を検出して自動判別します：

### 日本株の特徴
- ヘッダーに「銘柄コード」カラムが存在
- 数字のみの銘柄コード（4-5桁）

### 米国株の特徴
- ヘッダーに「国内約定日」カラムが存在
- 銘柄名に取引所名（"New York Stock Exchange", "NASDAQ"）が含まれる
- アルファベットのティッカーシンボル

## 実際のSBI証券CSVファイル

以下のファイル名形式に対応しています：
- 日本株: `SaveFile_000001_000040.csv`
- 米国株: `yakujo20260202112222.csv`

## データの永続化

インポートしたデータはブラウザのIndexedDBに保存されるため、ページをリロードしても保持されます。
全てのデータを削除したい場合は、トレード一覧の「全て削除」ボタンを使用してください。
